# glem0.github.io

Small web apps, served by GitHub Pages at https://glennmcgui.re/

- [gue-course-map](gue-course-map/) — the GUE class schedule on a world map
- [dive-nsw](dive-nsw/) — NSW dive sites and conditions
- [apple-cve](apple-cve/) — searchable index of Apple security releases

Each app is a self-contained folder with its own index.html.
